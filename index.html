¿Que vas a lograr?

  • Aprenderás a hacer maquetado con HTML

  • Uso mínimo viable de GIT y GitHub para desarrolladores
  • Aprenderás como crear "Scripts" para darle vida a tu página web
  • Cuales son los gatillos mentales que toda página de ventas debe tener

  • Ya no vas a necesitar pagar por un constructor de software, por que vas a saber como hacer tus páginas a puro código
  • Al finalizar sabrás como hacer un diseño front-end para lograr conversiones y adaptar tu página a cualquier rubro
  • Podrás imitar páginas web ganadoras que venden MILES de dólares todos los días
  • Los archivos que aprenderás a crear a base de código te van a quedar como plantilla para que puedas usarlas y/o venderlas como servicio
